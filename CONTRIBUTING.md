# Contributing to Task Master

Thank you for your interest in contributing!

## Setup

```bash
git clone https://github.com/sagar0163/task-master.git
cd task-master

# Open in browser
open index.html
```

## Making Changes

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make improvements
4. Commit: `git commit -m "feat: add new feature"`
5. Push to main: `git push origin main`

## Ideas for Contributions

- 💾 Local storage persistence
- 📅 Due dates & reminders
- 🏷️ Tags & categories
- 👥 Team collaboration
- 📊 Progress tracking
- 📤 Export functionality
- 🎨 Themes & dark mode
- 📱 Mobile responsive

## Tech Stack

- Vanilla JavaScript
- HTML5/CSS3
- Local Storage API
