## Features

- ✅ Create, edit, delete tasks
- ✅ Mark tasks as complete
- ✅ Priority levels (High, Medium, Low)
- ✅ Due dates with overdue tracking
- ✅ Project organization with colors
- ✅ Search and filters
- ✅ Statistics dashboard
- ✅ LocalStorage persistence

## Quick Start

```bash
# Simply open in browser - no server needed!
open index.html
```

Or serve locally:

```bash
npx serve .
```

## Tech Stack

- Vue.js 3 (Composition API)
- TailwindCSS
- localStorage

## Usage

1. Click **Add Task** to create a new task
2. Set **priority** and **due date**
3. Assign to a **project** for organization
4. Click the **checkbox** to complete tasks
5. Use **filters** to find specific tasks

## Keyboard Shortcuts

- Type to search tasks instantly
- Use dropdown filters for priority/status

## Data

All data is stored in your browser's **LocalStorage** - no account needed!
